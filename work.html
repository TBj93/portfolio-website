Movie API

This Api is the basis for the 2 projects below. It is their backend code. It enables to store data for user registration and detailed information about movies, genre and directors. The referring mongoDB database can be found on Cloud.MongoDB.com(Atlas) The API is hosted on heroku and thereby made available for the 2 clients working with it.

Myflix-React-Client

This is a working client rendering a database of movies. The movies are being displayed in cards using react-bootstrap and contain a movie image, fetched by an external URL, as well as information about the genre and its director. This client uses axios to fetch movies from a database, that is deployed on heroku Link. It's interactive and this client also provides the possiblity to create a user account(by clicking the register button), stored on the database.
